# Pudoku: sudoku web interface written in python

## With using :

* [sudokulib](http://pypi.python.org/pypi/sudokulib)
* [Twitter bootstrap](https://github.com/Twitter/bootstrap)


## How to use

```bash
$ python pudoku.py
$ python pudoku.py --port 789
```
